Introdução ao Pacote dplyr

Elisângela C. Biazatti, Douglas Vinícius, Jossivana Macedo
Slides Materiais Apostila
dplyr

O dplyr é uma ótima ferramenta para realizar manipulação de dados. Isso torna seu processo de análise de dados muito mais eficiente. Melhor ainda, é bastante simples aprender e começar a se aplicar imediatamente ao seu trabalho! Muitas vezes, com apenas algumas linhas de código elegantes, seus dados ficam muito mais fáceis de dissecar e analisar.
